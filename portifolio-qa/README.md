# Portfólio QA — Testes Manuais

Este diretório contém exemplos de testes manuais realizados em sites públicos.

## Testes realizados

### Teste Funcional — Mercado Livre

Objetivo: validar a funcionalidade de busca.

Cenários testados:

- busca por "notebook"
- validação dos resultados
- aplicação de filtros
- ordenação por menor preço
- busca inexistente "abcd1234"

---

### Teste de Interface — Amazon

Objetivo: validar elementos visuais da interface.

Itens verificados:

- barra de busca
- menu principal
- botão "Contas e Listas"
- dropdown do menu
- banners e carrossel

---

### Teste de Formulário — LinkedIn

Objetivo: validar comportamento do formulário de cadastro.

Cenários:

- email inválido
- campos vazios
- senha curta
- senha sem números
- senha sem letras

---

### Teste de Login — Magazine Luiza

Objetivo: validar comportamento do login.

Cenários:

- email correto + senha incorreta
- email inválido
- senha vazia
- email vazio
- mostrar senha
